# AWS Resume Site

## Overview

A full-stack resume website built with React and hosted on AWS.
Designed to demonstrate cloud infrastructure, CI/CD, and frontend skills.

## Tech Stack

- React
- AWS S3
- CloudFront
- Route 53
- GitHub Actions (CI/CD)

## Features

- Responsive resume layout
- Static site hosting on AWS
- Automatic deployment on push to main

## How to Run Locally

npm install
npm start
